# HotelBookingSystem
Hotel Booking Application

## Getting hotel-api
```
git clone https://github.com/KalyaniBothra/HotelBookingSystem.git
```

## Change the directory to HotelBookingSystem/src
```
cd HotelBookingSystem/src
```
## Install dependencies
```
npm install
```
## Starting the api
```
npm run start
```
